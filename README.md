# Employee-Payroll-System-Dashboard-Power-BI:                                                                                                                   The 
Employee Payroll System Dashboard is an interactive reporting solution developed using Power BI to analyze employee salary details, deductions, attendance, and department-wise compensation. The objective of this project was to simplify payroll analysis, reduce manual effort, and provide HR teams with clear and actionable insights.

The dataset, sourced from CSV/Excel files, includes employee information, attendance records, salary components, and deduction details. Data cleaning and transformation were performed using Power Query, where inconsistencies were resolved, missing values were handled, and data was structured for efficient analysis.

A well-defined data model was created by establishing relationships between employee, attendance, and payroll tables. Using DAX, key metrics such as total salary, net pay, deductions, and attendance-based calculations were developed to support detailed payroll analysis.

The dashboard includes multiple report pages such as Payroll Overview, Department Analysis, and Employee Details. It features interactive visualizations like KPI cards, bar charts, and tables to present salary distribution, deduction trends, and department-wise compensation. Slicers and filters were added for dynamic data exploration, along with page navigation using buttons and bookmarks to enhance user experience.

This dashboard enables HR teams to quickly monitor payroll performance, identify discrepancies, and analyze salary trends across departments. As a result, it reduced manual reporting effort by approximately 30% and improved accuracy and efficiency in payroll management. The solution supports faster and more informed decision-making by providing a centralized and easy-to-use platform for payroll insights.
